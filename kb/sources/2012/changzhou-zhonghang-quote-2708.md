# 常州中航大酒店幕墙与外装修工程造价估算(2708 万)

## 概述

常州中航大酒店幕墙与外装修工程造价汇总表 + 综合单价分析明细,总造价 **2708 万元**(27085372.76
元)。涵盖 7 大子项:主楼玻璃幕墙及石材包柱系统 10411m²、开放式花岗岩背栓挂板 8145m²、屋顶
钢结构装饰格栅 1772m²、裙楼石材窗及百叶 531m²、裙楼隐框玻璃幕墙 2508m²、裙楼雨篷 582m²、
防雨百叶 327m²。

## 关键信息

- 项目: 常州中航大酒店幕墙与外装修工程
- 类型: 造价估算(投标报价底价)
- 总造价: **2708 万元**
- 关键决策:
  - 主楼玻璃幕墙 + 石材包柱组合 1161.78 元/m²(含 LOW-E 中空 + 花岗岩 + 钢结构)
  - 开放式花岗岩背栓挂板 1080.5 元/m²(纯石材系统,无玻璃)
  - 屋顶钢结构装饰格栅 1403.45 元/m²(单价最高,钢结构为主)
